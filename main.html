Minor Projects

Heightmap Board Generator

While experimenting with the concept of a digital board game, I found myself interested in the topic of procedural generation. Before long I was experimenting with creating that would enable quick, iterative design.

I created a script that would take an image and output an arrays of hexagonal objects, using the value of each pixel to determine the height at which each object should be outputted.

Python Script Automation

I commonly find myself seeking out ways to lower friction in my day-to-day by automating tasks like transcription and data entry. This image shows the opening lines of a script that I wrote. It parses markdown into JSON ready for import into into FoundryVTT, a virtual tabletop software built using web technologies.
